## About The Project



This project was built to demonstrate the mathematical underpinnings of **Multiple Linear Regression**. While libraries like Scikit-Learn make implementation easy, understanding the core algorithms is crucial for any data scientist.

In this project, I implemented the algorithm in two ways:
1.  **Using Scikit-Learn:** To establish a baseline for performance and accuracy.
2.  **From Scratch:** Creating a custom class `NewMLR` that utilizes the **Normal Equation** $(X^T X)^{-1} X^T Y$ to solve for coefficients without relying on pre-built ML libraries.

Both models were trained on a synthetic dataset generated with controlled noise, and the results show that the custom implementation achieves identical coefficients and intercepts to the optimized library version.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need Python installed on your system. You will also need the following libraries.
* pip
  ```sh
  pip install numpy pandas matplotlib scikit-learn

### Installation 

1. Clone the repo

```sh
git clone https://github.com/aadityasinghweb2005/Project-10-Multiple-Linear-Regression.git
```
Navigate to the project directory

2. Navigate to the project Directory 
```sh
cd Project-10-Multiple-Linear-Regression
```
Install the dependencies (if you haven't already)

3. Install the dependencies (if You haven't already)

```sh

```
